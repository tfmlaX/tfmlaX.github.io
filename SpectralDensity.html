Bath Spectral Density

Hamiltonian¶

Lets consider a system with a non-specified free Hamiltonian $\hat{H}_S$ interacting with a bosonic bath of free Hamiltonian $\hat{H}_B = \sum_k \omega_k \hat{a}^\dagger_k\hat{a}_k$ through an interaction Hamiltonian linear in bath operators $\hat{H}_{\text{int}} = \hat{O}\sum_k ( g_k\hat{a}_k + \text{h.c.})$.

where $\hat{a}_k$ is the $k^\text{th}$ bath mode's annihilation operator and $g_k$ is the coupling strength between the system and the $k^\text{th}$ bath mode.

This interaction Hamiltonian can be expressed in the interaction picture as $\hat{H}_\text{int} = \hat{O}\sum_n (g_n\hat{a}_n\mathrm{e}^{-\mathrm{i}\omega_n t} +g_n^{*}\hat{a}_n^\dagger\mathrm{e}^{+\mathrm{i}\omega_n t})$ .

There we can define the hermitian bath operator $\hat{B}(t)$: $$\hat{B}(t) = \sum_n (g_n\hat{a}_n\mathrm{e}^{-\mathrm{i}\omega_n t} +g_n^{*}\hat{a}_n^\dagger\mathrm{e}^{+\mathrm{i}\omega_n t}).$$

Bath Correlation Function¶

The associated correlation function is $$C_B(t,t') = \langle \hat{B}(t)\hat{B}(t')\rangle -\underbrace{\langle \hat{B}(t) \rangle\langle \hat{B}(t') \rangle}_0$$ where $\langle \circ \rangle = \text{tr}[\circ \hat{\rho}_B]$ with $\hat{\rho}_B = \frac{1}{Z}\exp(-\beta\hat{H}_B)$.

All calculations done, one finds that $$C_B(t-t') = \sum_{i} |g_i|^2\left[(2n_B(\omega_i) + 1)\cos\left(\omega_i(t-t')\right) - \mathrm{i}\sin\left(\omega_i(t-t')\right)\right]$$ where $n_B$ is the Bose-Einstein distribution.

The bath correlation function is complex with the property $C_B^*(\tau) = C_B(-\tau)$, which implies from the properties of the Fourier Transform that the Power spectral Density is real (luckily, because it's also the modulus squared of the bath operator's Fourier transform).

The correlation function can be decomposed into its real part and its imaginary part. Property relating complex conjugasrion and time-symmetry is also true for $\Re[C_B(\tau)]$ and $\mathrm{i}\Im[C_B(\tau)]$ whose Fourier transforms are therefore real. The real part of the correlation function being real, its Fourier transform is symmetric. Conversely the imaginary part of the correlation function has an anti-symmetric Fourier transform.

Therefore, $S_B(\omega) = S_B^S(\omega) + S_B^A(\omega)$ with, by direct calculation: $$S_B^S(\omega) = 2\pi\sum_n |g_n|^2 (2n_B(\omega_n)+1)\frac{1}{2}(\delta(\omega- \omega_n) + \delta(\omega + \omega_n))\ ,$$ $$S_B^A(\omega) = 2\pi\sum_n |g_n|^2 \frac{1}{2}(-\delta(\omega- \omega_n) + \delta(\omega + \omega_n))\ .$$

Because the bath energy is only physically defined for positive frequency we introduce the physical power spectrum $$ \mathcal{S}_B(\omega) = \Big\{ \begin{matrix} &S_B(\omega) &\text{for}\ \omega > 0\\ &0 & \text{for}\ \omega < 0 \end{matrix} \ \ .$$

Hence $$\mathcal{S}_B(\omega) = 2\pi n_B(\omega)\sum_n |g_n|^2\delta(\omega-\omega_n) = 2\pi n_B(\omega)J(\omega)$$ where we define the bath spectral density $$J(\omega) = \sum_n |g_n|^2\delta(\omega-\omega_n)$$ as the part of the physical power spectrum that doesn't depends on temperature.

The last equation thus states that the bath power spectrum is the bath spectral density modulated by the thermal population $n_B(\omega)$.

With this definition of $J(\omega)$, we can rewrite the bath correlation function in the more familiar integral form

$$C(\tau) = \int_0^{\infty} \mathrm{d}\omega J(\omega)\left[\coth\left(\frac{\hbar\omega}{2k_BT}\right)\cos\left(\omega\tau\right) - \mathrm{i}\sin\left(\omega\tau\right)\right]\ .$$

We can already see that for different spectral densities $J(\omega)$ the bath correlation function is going to have different behaviours, and especially different timescales for the bath correlation time $\tau_B$.

If all the bath modes couple to the bath the same way, i.e. $J(\omega) = J_0$, at zero temperature the correlation function is a Dirac delta function. Thus there are only instantaneous correlations into the bath. It is a limit case of what is called a Markovian behaviour.

Flat Spectral Density¶

To appxoximate the limit case of a perfectly flat spectral density at $T=0$ K, we look at the bath correlation function for a rectangular spectral density centered on $\omega_0$ with a width $\Delta\omega$.

$$C(\tau) = J_0 \Delta\omega \text{sinc}(\frac{\Delta\omega\tau}{2})\exp(-\mathrm{i}\omega_0\tau) $$

The characteristic correlation time of the bath correlation function (corresponding to the first zero of the $\text{sinc}$) is : $\tau_B = \frac{2\pi}{\Delta\omega}$.

Note that $\lim_{\Delta\omega \to \infty} C(t) = \delta(t)$.

f = plt.figure(figsize=(15,5))
ax1 = f.add_subplot(121, xlabel="t", ylabel="|C(t)|")
ax2 = f.add_subplot(122, xlabel="ω",ylabel="J(ω)")


J0 = 1
C = []
t = arange(-0.01,1,0.01)
DeltaOmega = [5,10,100]
legend = []

for Dw in DeltaOmega:
    C = J0*Dw*pi*absolute(sinc(Dw*t/(2*pi)))
    ax1.plot(t,C)
    legend.append(r'Δω = {}'.format(Dw))
ax1.legend(legend)


J = []
omega = arange(100,400,0.1)
omega0 = 250

for Dw in DeltaOmega:
    J = [J0*int(om <= (omega0+Dw) and om >= (omega0-Dw)) for om in omega]
    ax2.plot(omega,J)
    legend.append(r'Δω = {}'.format(Dw))
ax2.legend(legend)


plt.show()

Lorentzian Spectral Density¶

The Lorentzian Spectral Density is peaked around a specific angular frequency $\omega_0$ amd has a bandwidth $\alpha$.

$$ J(\omega) = \frac{\alpha^2 J_0}{(\omega - \omega_0)^2 + \alpha^2}$$$$ C(\tau) = \pi\alpha J_0 \exp(-\alpha\tau)\exp(-\mathrm{i}\omega_0\tau) $$

The caracteristic correlation time of the bath correlation function is $\tau_B = \frac{1}{\alpha}$.

If the bandwidth is narrow only a few frequencies are selected and the correlation function has a large correlation time, with the limit case of the Dirac delat spectral density leading to an flat correlation function (i.e. a infinite correlation time).

On the other hand, the larger the bandwith, the flatter the spectral density and the more peaked the correlation function.
